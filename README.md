## Getting Started

### Prerequisites

- Flutter (2.0 or higher)
- Dart SDK
- Android Studio / VS Code

### Installation

1. Clone the repository
```console
git clone https://github.com/toBeRoman/bazicab-flutter.git
```

2. Navigate to project directory
```console
cd bazicab-flutter
```

3. Install dependencies
```console
flutter pub get
```

5. Run the app

```console
flutter run
```

## Dependencies

- google_maps_flutter: ^2.5.3
- intl: ^0.19.0
- shimmer: ^3.0.0
